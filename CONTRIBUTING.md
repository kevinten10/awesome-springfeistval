# 贡献指南

感谢您对 **Awesome Spring Festival** 项目的关注！欢迎推荐春节主题的优质开源项目。

---

## 📋 收录标准

### ✅ 我们会收录

- **主题相关**：春节、新年、元宵、农历、生肖等中国传统节日相关的项目
- **开源项目**：在 GitHub 上有公开代码仓库
- **内容完整**：有 README、代码、使用说明
- **活跃维护**：最近 2 年有更新，或是经典项目
- **描述清晰**：项目用途和技术栈说明明确

### ❌ 我们不会收录

- 仅含营销链接或无实际代码的项目
- 已失效或删除的仓库
- 含有违法、侵权或不当内容的项目
- 纯商业项目（除非有开源组件）

---

## 📝 推荐项目格式

### 通过 Issue 推荐

点击 [提交项目](../../issues/new?template=project_submission.md)，填写以下信息：

```markdown
## 项目信息

**项目名称**：
**GitHub 链接**：
**项目描述**（一句话）：
**主要技术栈**：
**在线演示**（可选）：
**分类**（视觉效果/农历日历/春联祝福/游戏娱乐/实用工具）：
```

### 通过 Pull Request 推荐

1. Fork 本仓库
2. 编辑 `README.md`
3. 按以下格式添加项目：

```markdown
| [项目名](https://github.com/用户名/项目名) | 简短描述 | 技术 | ![GitHub Repo stars](https://img.shields.io/github/stars/用户名/项目名?style=social) | [🔗](演示链接) |
```

4. 提交 Pull Request

---

## 🔍 项目验证

为确保项目质量，我们可能会检查：

- [ ] 项目链接有效（404 检查）
- [ ] 描述准确（基于 README）
- [ ] 技术栈正确
- [ ] 项目活跃度（最近提交时间）
- [ ] 是否有安全/合规问题
